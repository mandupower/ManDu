# EverHisGrace Score Library

Welcome to the EverHisGrace Score Library.

This repository contains lead sheets, sheet music, and educational music resources created for churches, families, schools, and worship communities.

## Available Resources

- Lead Sheets
- Piano Scores
- Children's Worship Songs
- Choir Arrangements
- Educational Music Materials

## Current Scores

- Together We Shine
- Jesus Holds My Hand
- One Tree

More scores will be added over time.

## Usage

You may download, print, and use these materials for:

- Church worship services
- Sunday school
- Children's ministry
- Family worship
- Educational purposes

Please do not resell or redistribute modified versions without permission.
